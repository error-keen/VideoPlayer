# 一、安装K-Lite

K-Lite 是一个万能解码器，它可以扩展播放器功能，使其能解码更多格式。

# 二、QT版本要求

1. qt5的版本都可以，编译器使用mingw32

2. 不要把代码放在桌面，路径不能包含中文。
